### Overview

A responsive, professional portfolio website built to showcase my legal tech projects, automation workflows, and professional background. Designed with clean modern aesthetics and built using vanilla HTML, CSS, and JavaScript.

---

### Features

- **Hero Section** - Professional introduction with animated typing effect
- **About Me** - Detailed professional bio highlighting legal + tech expertise
- **Projects Showcase** - Grid layout displaying featured projects with images
- **Tech Stack** - Visual badges showing core technologies
- **Experience Timeline** - Career highlights and ventures
- **Contact Section** - Email, LinkedIn, and social links
- **Responsive Design** - Fully mobile-friendly across all devices
- **Dark/Light Theme** - Toggle between themes for better readability

---

### Pages / Sections

1. **Home** - Hero banner with CTA
2. **About** - Professional background and story
3. **Projects** - Featured work with descriptions
4. **Skills** - Technical and legal expertise
5. **Experience** - Timeline of roles and ventures
6. **Contact** - Direct outreach options

### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Padmasri792/portfolio-website.git
   ```

2. **Navigate to the folder**
   ```bash
   cd portfolio-website
   ```

3. **Open in browser**
   ```bash
   open index.html
   ```

---

### Customization

To personalize this template:
- Edit **index.html** to update text and sections
- Modify **styles.css** for colors and layout
- Replace images in **assets/images/**
- Update links to your social profiles
